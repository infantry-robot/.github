## 2026赛季步兵车组赛季规划
### 车组成员

吴泓宇 步兵车组组长

王潇霏 曹洪川 王楷尧 陆沛奇 普步机械

孙庆洋 陈思翔 平步机械

李易霖 景宇航 普步电控

**步兵车组根据现有技术积累，给出了赛季总体规划：本赛季步兵车组希望完成一辆下供弹步兵、一辆腰供弹步兵和一辆串联腿平步的制作，为战队补齐RM赛场中现有的主流步兵车型的技术栈，在26赛季完成步兵技术的全面积累，为战队此后赛季的步兵升级迭代打下一个真正的基础。**
